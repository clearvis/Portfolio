Google´s Data Highlighter Tool

Written by James Sacci on November 21, 2013.

Data Highlighter is a tool for webmasters, or website owners with a Google Webmaster Tools account, that teaches search engines about the pattern of structured data on your website. Data Highlighter offers an easy to use interface with form fields used to tag the data fields on a website. Search engines can then present your data more meaningful ways on their search results pages.

For website owners without knowledge of HTML, Data Highlighter is the perfect solution for giving their website a SEO boost.
